## **🗒️**目录

main分支包含所有的学科，各个学期的请在左上角切换分支即可。

克隆指定分支, 如克隆"大二上"分支, 可采用如下命令：
```shell
git clone -b 大二上 https://github.com/LeoMooreCST/CQUCS-Study.git
```
